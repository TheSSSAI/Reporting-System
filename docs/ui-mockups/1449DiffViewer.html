Diff Viewer

A component for comparing two blocks of text, mimicking a code diff tool.

Side-by-Side (Responsive)

This view shows the original and changed versions in two columns on wide screens, and stacks them on screens narrower than 900px.

Original (Version 3)
1{2  "reportName": "Weekly Sales",3  "format": "CSV",4  "schedule": "0 8 * * 1",5  "recipients": [6    "management@example.com"7  ]8}
Changed (Version 4)
1{2  "reportName": "Weekly Sales",3  "format": "PDF",4  "schedule": "0 8 * * 1",5  "delivery": {6    "type": "email",7    "recipients": [8      "management@example.com",9      "sales-team@example.com"10    ]11  }12}

Inline

This view shows changes in a single, unified column.

Changes from Version 3 to Version 4
1{2  "reportName": "Weekly Sales",3  "format": "CSV",3  "format": "PDF",4  "schedule": "0 8 * * 1",5  "recipients": [6    "management@example.com"7  ]5  "delivery": {6    "type": "email",7    "recipients": [8      "management@example.com",9      "sales-team@example.com"10    ]11  }12}

Component Variations

Variant Name : "side_by_side"

Displays two versions of the content in two parallel columns for direct comparison.

Original
1"color": "blue",2"size": "medium"
Changed
1"color": "red",2"size": "medium"

Variant Name : "inline"

Displays changes in a single, unified view, interleaving removed and added lines.

Changes
1"color": "blue",1"color": "red",2"size": "medium"
