Description

A layout component that displays the title of a page, an optional subtitle, and a set of primary/secondary actions. It serves as the primary contextual anchor for a screen.

Design System Alignment

Uses typography tokens (h2, body) for text, spacing tokens for padding and gaps, and color tokens for text and borders. It directly incorporates the 'Button' atomic component for its action slots.

Responsive Specifications

Mobile Adaptations

  • Switches to a vertical layout on screens < 768px.
  • Action buttons stack below the title and subtitle.
  • Buttons in the action group can optionally be configured to grow and fill the available width.

Tablet Adaptations

  • Maintain desktop horizontal layout.
  • Ensure sufficient spacing between title and action buttons.

Desktop Enhancements

  • Optimized horizontal layout with title on the left and actions on the right.
  • Clear visual hierarchy between title, subtitle, and actions.

Accessibility Features

Keyboard Support

The action buttons within the header are focusable and activatable via keyboard.
