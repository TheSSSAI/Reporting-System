Description

A wrapper component that logically groups a label, an input control (like TextInput, SelectDropdown, Checkbox), and an optional error or help message. It manages the layout and state (e.g., error) for these elements as a single unit, ensuring consistent spacing and accessible labeling.

Design System Alignment

Utilizes design system tokens for spacing (var(--space-xs), var(--space-md)), typography (var(--font-size-sm)), and semantic colors (var(--color-text-primary), var(--color-text-secondary), var(--color-danger-500)) to ensure consistency with the overall visual language.

Responsive Specifications

Mobile Adaptations

  • Default vertical stacking of label, input, and message works well on mobile.
  • Increased bottom margin for better separation in single-column layouts on screens < 768px.

Tablet Adaptations

  • Maintains the standard vertical layout, which provides clear and readable forms on tablets.

Desktop Enhancements

  • The component maintains a clean vertical rhythm, which is optimal for form scannability even on large screens.

Accessibility Features

Keyboard Support

Standard keyboard navigation is inherited by the child input elements (TextInput, Select, Checkbox).
