Description

A component that allows users to select files for upload, either by clicking a button or via a drag-and-drop area. It provides visual feedback for various states like uploading, success, and error.

Design System Alignment

Utilizes design system tokens for colors (primary, success, error, neutrals), spacing, border radius, and typography. It also composes atomic components like Button, Icon, and Spinner.
