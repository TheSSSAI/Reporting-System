Responsive Specifications

Mobile Adaptations

  • The table container (.data-table-scroll-container) becomes horizontally scrollable on viewports smaller than 1280px to accommodate wide tables without breaking the page layout.

Tablet Adaptations

  • The horizontal scrolling behavior applies to tablet viewports as well.

Desktop Enhancements

  • Full table visibility without horizontal scrolling on wider screens.
  • Hover effects on rows and sortable headers.

Accessibility Features

Keyboard Support

Interactive elements like checkboxes and sortable headers are focusable via Tab key. Checkboxes are toggled with Space. Headers can be activated with Enter/Space.
