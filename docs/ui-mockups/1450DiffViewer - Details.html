Component Name

DiffViewer

Component Type

organism

Description

A specialized component that displays a side-by-side or inline comparison of two text blocks, typically code or JSON. It highlights added, removed, and unchanged lines. It is designed to visually mimic the experience of the Monaco Editor's diffing capabilities.

Design System Alignment

Uses design system tokens for fonts (monospace), colors (semantic error/success for diffs, neutrals for background/text), and spacing.

Responsive Specifications

Mobile Adaptations

  • On viewports < 768px, the side-by-side variant stacks vertically, effectively behaving like an inline view to prevent horizontal overflow of the component itself.

Tablet Adaptations

  • Tablet viewports may show the stacked view or side-by-side view depending on width and orientation. The breakpoint is set at 768px.

Desktop Enhancements

  • Defaults to a full side-by-side comparison, making it easy to track changes across two versions of a file.

Accessibility Features

Keyboard Support

The component's content can be scrolled using standard keyboard controls (arrow keys, Page Up/Down).
