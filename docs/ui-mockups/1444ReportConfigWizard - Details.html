Description

A multi-step guided wizard for creating and editing a report configuration. It breaks down the complex process into logical steps, ensuring all required information is gathered in a user-friendly manner. It handles metadata, data source selection, transformations, output formats, delivery, and scheduling.

Design System Alignment

This organism composes several molecular and atomic components like FormGroup, SelectDropdown, Button, and Alert. It utilizes the design system's spacing, typography, and color tokens to create a cohesive layout and visual hierarchy.

Responsive Specifications

Mobile Adaptations

  • The wizard switches to a full-width, column layout.
  • The stepper becomes a horizontal, scrollable list at the top, instead of a vertical sidebar.

Tablet Adaptations

  • The layout may switch to the vertical mobile layout on portrait tablets, but maintains the desktop layout on landscape tablets.

Desktop Enhancements

  • A two-column layout with a persistent vertical stepper on the left and the main content area on the right.
  • Ample whitespace and clear separation of content sections.

Accessibility Features

Keyboard Support

All interactive elements (buttons, inputs, stepper items) are keyboard focusable and activatable.

Screen Reader Support

The stepper uses an ordered list for semantic structure. The active step is indicated with `aria-current='step'`. All form controls have associated labels. Error messages are linked with `aria-describedby`.

Focus Management

Focus is managed logically as the user progresses through the steps.

Aria Attributes

  • aria-current='step'
  • aria-describedby

Color Contrast

All text and UI elements meet WCAG 2.1 AA contrast requirements.
