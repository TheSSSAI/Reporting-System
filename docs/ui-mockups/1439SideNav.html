Dashboard

This is the main content area. The side navigation is an organism component that provides the primary means of navigating between major sections of the application.

On desktop viewports wider than 1280px, it is expanded by default but can be collapsed. Below 1280px, it automatically collapses to an icon-only view to save space. A checkbox toggle is provided here for desktop demonstration purposes.
