Responsive Specifications

Desktop Layout

  • Modal has a max-width of 960px.
  • DataTable columns have defined widths for optimal readability.
  • Diff viewer uses a side-by-side layout.

Tablet Adaptations

  • Modal width adjusts to fit the viewport with appropriate margins.
  • Font sizes and spacing remain consistent.

Mobile Adaptations

  • Modal takes up most of the screen width.
  • Modal body becomes scrollable if content overflows.
  • Diff viewer may stack or become horizontally scrollable to fit content.
