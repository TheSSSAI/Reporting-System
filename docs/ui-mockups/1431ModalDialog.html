Modal Dialog

This component displays content in a layer above the main application. It is used for confirmations, forms, or important information that requires user attention.

Note: This demonstration uses a CSS-only checkbox hack to show/hide the modals. A production implementation would use JavaScript to manage state, focus trapping, and keyboard interactions (like the Escape key).

Confirmation Dialog

Used to confirm a user's intent to perform a potentially destructive or irreversible action.

Informational Dialog

Used to present important information to the user that must be acknowledged.

Form Dialog

Used to host a short, self-contained form for actions like creating a new item or editing settings.
